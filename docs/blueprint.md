# **App Name**: Rahul Raj Portfolio

## Core Features:

- Dynamic Content Rendering: Display resume content including work experience, projects, and skills.
- Interactive Project Showcase: Provide interactive previews of the featured projects, sourced directly from provided links, allowing users to explore code repositories or live applications with ease.
- AI-Powered Personalization Tool: Employ a tool to tailor content for different job applications or audience profiles using the information provided. This would enable highlighting relevant skills and experience based on user input, generating different portfolio texts or skill sets that better suit each audience.
- Contact Form Integration: Enable potential employers or collaborators to directly contact Rahul Raj through the portfolio website, using a form.
- Responsive Design: Ensure the portfolio is accessible and visually appealing across various devices (desktops, tablets, and mobile phones) through a responsive design.

## Style Guidelines:

- Primary color: Dark navy blue (#243A73), conveying professionalism.
- Background color: Very light grey (#F0F4F8), providing a clean backdrop.
- Accent color: Soft coral (#F08080), used sparingly for highlights and interactive elements.
- Body and headline font: 'Inter' sans-serif, for a clean, modern, and readable appearance.
- Clean and structured layout with clear sections for different resume aspects (experience, skills, projects). Prioritize readability and easy navigation.
- Use simple, professional icons for skills and contact information. Consistent style to maintain visual harmony.
- Subtle transitions and animations to improve user engagement. Avoid overly distracting animations; keep it professional and smooth.
